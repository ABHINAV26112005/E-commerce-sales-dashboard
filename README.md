📌 Project Overview

This project is an interactive E-Commerce Sales Dashboard built entirely in Microsoft Excel. The dashboard transforms raw sales data into meaningful insights, helping businesses track performance, monitor key metrics, and make data-driven decisions.

The goal of this project was to practice data analysis, dashboard design, and business reporting using Excel's powerful visualization tools.

🎯 Objectives

Analyze e-commerce sales performance

Monitor revenue and order trends

Track customer ratings and delivery performance

Identify top-performing categories and products

Create an interactive dashboard for business decision-making

📈 Key Metrics

Total Revenue

Total Orders

Quantity Sold

Average Customer Rating

Average Delivery Time
